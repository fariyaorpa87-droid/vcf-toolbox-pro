使用说明

这是纯单机 Electron 桌面版。支持上传 TXT/CSV 或直接粘贴号码。

打包方法：上传到 GitHub 后，进入 Actions，运行 Build Windows EXE，下载 Artifacts。

安装版：Setup.exe；免安装版：Portable.exe。
